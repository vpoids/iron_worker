#Iron worker projects

##Getting Started

1. Install composure:

```ShellSession
curl -sS https://getcomposer.org/installer | php -d detect_unicode=Off
```

2. Install php-opencloud

```ShellSession
php -d detect_unicode=Off composer.phar require rackspace/php-opencloud:dev-master
```
